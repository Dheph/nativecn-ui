## Overview

`Switch` is a React Native component designed for creating a switch or toggle control. It's built on top of React Native's [Switch](https://reactnative.dev/docs/switch). It can be customized with Tailwind classes through the `className`.

## Installation

Include the `Switch.tsx` file in your React Native project. [File Link](https://github.com/Mobilecn-UI/nativecn-ui/blob/main/components/Switch.tsx)

## Properties

- `className?: string (optional)` - The classes for the switch.
- See React Native's [Switch props](https://reactnative.dev/docs/switch#props)

## Usage

### Basic Usage

Here's how you can use the `Switch` component:

```jsx
const [isEnabled, setIsEnabled] = useState(false);

return <Switch onValueChange={setIsEnabled} value={isEnabled} />;
```
